## Analise-de-Dados
Repositório dedicado ao estudo e prática de Análise de Dados, contendo exemplos de limpeza, tratamento e análise de dados utilizando Excel, SQL e Power BI. Inclui manipulação de dados brutos (CSV), criação de métricas e KPIs, tabelas dinâmicas, visualizações e análises voltadas à tomada de decisão, com foco em boas práticas e fundamentos da área.

## Estrutura do Projeto
- Base Original: dados brutos sem tratamento
- Base Tratada: dados limpos e padronizados
- Análise: cálculos e métricas
- Meta Mensal: regras de negócio
- Dashboard: visualização final

## Objetivo do Projeto
Analisar dados de vendas com o objetivo de identificar o desempenho por vendedor,
produto e período, acompanhando metas mensais e indicadores de faturamento,
auxiliando na tomada de decisão estratégica.

## Principais KPIs
- Faturamento total
- Faturamento por vendedor
- Faturamento por produto
- Meta x Realizado
- Ticket médio
- Evolução mensal de vendas

## Tecnologias Utilizadas
- Excel
- Tabelas Dinâmicas
- Fórmulas
- KPIs
- Análise de Dados
- Visualização

## Conclusão
O projeto demonstra o processo completo de análise de dados,
desde a organização da base até a geração de insights por meio
de dashboards e indicadores de desempenho.
